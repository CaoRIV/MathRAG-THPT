# MathRAG THPT Design System

The interface follows the UI-UX Pro Max AI-native recommendation, adapted for a serious Vietnamese education product.

## Foundations

- Primary: `#7C3AED`
- Accent: `#0891B2`
- Canvas: `#FAF9FF`
- Text: `#1E1B4B`
- Headings: Be Vietnam Pro
- Body: Noto Sans
- Mathematics: KaTeX

## Experience rules

- Keep the learning workspace calm and single-purpose.
- Distinguish answer, formula, evidence, citation, and recommendation blocks.
- Never use color as the only status indicator.
- Every async update uses a visible state and an ARIA live region.
- Clickable elements have stable hover feedback and visible keyboard focus.
- Motion is limited and disabled by `prefers-reduced-motion`.
- Validate layouts at 375, 768, 1024, and 1440 pixels.
